these are my dotfiles, managed by [chezmoi](https://www.chezmoi.io/)
## Installation

```sh
git clone https://github.com/chezmoi/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
chezmoi apply
```

## Updating

```sh
cd ~/.dotfiles
chezmoi apply
```
